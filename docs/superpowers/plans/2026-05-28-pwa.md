# PWA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Progressive Web App support so the dashboard is installable and works offline, without breaking the existing web build.

**Architecture:** Separate `vite.config.pwa.ts` extends the base config and adds `vite-plugin-pwa`. Normal `npm run dev/build` is unchanged. PWA build is opt-in via `npm run build:pwa`. Icons are generated from a single SVG source using `@vite-pwa/assets-generator`. Service worker is auto-generated by workbox — no hand-written SW code.

**Tech Stack:** vite-plugin-pwa 0.21+, @vite-pwa/assets-generator, Workbox (via plugin), Vite 6

### Task 1: Install dependencies, create PWA Vite config, add scripts

**Files:**
- Modify: `package.json`
- Create: `vite.config.pwa.ts`

- [ ] **Step 1: Install dependencies**

```bash
npm install --save-dev vite-plugin-pwa @vite-pwa/assets-generator
```

Expected: both packages appear in `package.json` devDependencies.

- [ ] **Step 2: Create vite.config.pwa.ts at repo root**

```ts
import { defineConfig, mergeConfig } from 'vite'
import { VitePWA } from 'vite-plugin-pwa'
import baseConfig from './vite.config'

export default mergeConfig(baseConfig, defineConfig({
  plugins: [
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: 'React Interview Dashboard',
        short_name: 'Interview',
        description: 'React interview prep and component showcase',
        theme_color: '#1e293b',
        background_color: '#0f172a',
        display: 'standalone',
        start_url: '/playGround-react/',
        scope: '/playGround-react/',
        icons: [
          {
            src: 'pwa-192x192.png',
            sizes: '192x192',
            type: 'image/png',
          },
          {
            src: 'pwa-512x512.png',
            sizes: '512x512',
            type: 'image/png',
          },
          {
            src: 'maskable-icon-512x512.png',
            sizes: '512x512',
            type: 'image/png',
            purpose: 'maskable',
          },
        ],
      },
      workbox: {
        globPatterns: ['**/*.{js,css,html,ico,png,svg,woff2}'],
        navigateFallback: '/playGround-react/index.html',
      },
    }),
  ],
}))
```

- [ ] **Step 3: Add scripts to package.json**

Open `package.json` and add to the `"scripts"` block:

```json
"dev:pwa":   "vite --config vite.config.pwa.ts",
"build:pwa": "tsc && vite build --config vite.config.pwa.ts && cp dist/index.html dist/404.html"
```

- [ ] **Step 4: Verify TypeScript passes**

```bash
npx tsc --noEmit
```

Expected: no output (clean).

- [ ] **Step 5: Commit**

```bash
git add vite.config.pwa.ts package.json package-lock.json
git commit -m "chore(pwa): install vite-plugin-pwa and create PWA Vite config"
```

- [ ] **Step 1: Update index.html**

Replace the entire file with:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>React Interview Dashboard</title>

    <meta name="theme-color" content="#1e293b" />
    <meta name="apple-mobile-web-app-capable" content="yes" />
    <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />
    <meta name="apple-mobile-web-app-title" content="Interview" />

    <link rel="apple-touch-icon" href="/playGround-react/apple-touch-icon-180x180.png" />
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
  </body>
</html>
```

Note: `<link rel="manifest">` is intentionally omitted — `vite-plugin-pwa` injects it automatically during `build:pwa`. In normal dev/build it's absent, which is correct.
